# Antigravity Project Hub

Bienvenido al centro de mando de tus proyectos gestionados por Rosa (Antigravity).

## Estructura
- `/projects`: Contiene los directorios individuales de cada proyecto/perfil.
- `backlog.md`: Seguimiento de tareas globales de Scrum.
- `history.md`: Registro de decisiones y progreso histórico.

## Workflow Recomendado
1. **Planificación**: Definir historias de usuario en `backlog.md`.
2. **Desarrollo**: Trabajar dentro de la carpeta del proyecto en `/projects`.
3. **CI/CD**: Sincronizar con GitHub y desplegar vía Cloud Build a GCP.
